# D2-T05 连续研究价格构建与反推校验门禁及合成验收框架

状态：

blocked pending raw and factor authorization via PR TBD

## 目标

建立 D2-T05 连续研究价格构建与反推校验的可执行门禁、contract、blocking report、
validator 和合成验收测试。本任务固定未来从 raw trading facts 与 adjustment factor
构造 `adj_open`、`adj_high`、`adj_low`、`adj_close` 前必须满足的字段、主键、as-of、
revision、反推校验、机械缺口隔离和质量检查。

本 PR 不完成真实 continuous research price construction，不生成真实
`d2.adjusted_market_prices`，不构造真实连续研究价格。

## 非目标

- 不导入公司行为；

- 不构造真实连续研究价格；
- 不生成 adjusted prices 或 `d2.adjusted_market_prices` 实体数据；
- 不写 DuckDB；

- 不计算 raw gap、adjusted gap、gap attribution、PCVT、状态、事件、标签、收益或回测；
- 不解锁 D2-T06。

## 契约边界

- `target_table` 固定为 `d2.adjusted_market_prices`；
- required output fields 与 primary key 必须匹配 D1-T03 / D2-T04 中已定义的
  adjusted price 契约；
- synthetic validation rows 可以同时包含 raw input fields、factor input fields 和 adjusted
  output fields，但不得被提交为真实数据产物；
- `identity_no_adjustment` 要求 `adjustment_factor == 1.0`，且 `adj_* == raw_*`；
- 合成 `multiplicative_factor` 规则要求 `adj_* == raw_* * adjustment_factor`；
- reverse check 必须能用 `adj_* / adjustment_factor` 反推回 raw OHLC；
- `factor_as_of_time` 必须存在，且不得晚于对应 `trading_date` 的 observation cutoff；
- 公司行为机械缺口不得被解释为普通市场跳空、趋势变化、释放、事件标签或 PCVT 状态；
- D2-T06 不因本 PR 自动解锁。

## 阻塞条件

当前 D2-T05 仍被以下条件阻塞：

- D2-T03 raw price materialization 仍为 `blocked_pending_source_authorization`；
- D2-T04 adjustment factor materialization 仍为 `blocked_pending_factor_source_authorization`；
- source terms 与正式 ingestion 授权未关闭；
- raw price、adjustment factor、factor as-of、revision policy、raw snapshot 和 manifest
  均未达到正式 construction 前置条件；
- 真实 continuous research price artifact、DuckDB 写入和 manifest 创建均未授权；
- D2-T06 gap attribution / price quality flag materialization 未授权。

## 验收标准

- contract 与 blocking report 通过 JSON Schema；
- contract 明确所有真实 continuous price、adjusted price、DuckDB 写入和 artifact 授权均为 false；
- blocking report 明确所有真实数据读取、生成、写入和 manifest flags 均为 false；
- validator 只验证内存对象或合成 fixture；
- 合成测试覆盖 identity、multiplicative factor、required fields、禁止字段、OHLC、factor
  正数、factor as-of、反推校验、词表 unknown、membership alignment、重复 primary key 和静默 false；
- README 不推进到 D2-T06；
- D2-T06 不因本 PR 自动解锁。
