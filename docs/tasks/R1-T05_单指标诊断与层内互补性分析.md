# R1-T05 单指标诊断与层内互补性分析

## 目标

本 task 是 GOV-T01 生效后的 R1 formal experiment。目标是对八项 active indicator 做单指标 raw/score 分布、层内 score 相关、q={0.10,0.20,0.30} threshold 2x2、individual hit duration 与 joint both-hit duration 诊断，并将每个 layer x W 标记为 `complementary_structure`、`redundancy_warning`、`construct_conflict_warning` 或 `insufficient_eligible_sample`。这些 status 仅为结构 warning，不构成指标删除、参数选择或 R2 冻结依据。

## 非目标

本 PR 不做 R1-T06 conditional retention、Lift 或 Delta，不做 R1-T07 滞后结构，不做 R1-T08/R1-T09 零模型或稳定性判定，不做状态线参数选择、layer-specific q、未来标签、未来收益、未来波动、释放方向、回测、组合、交易信号、因果解释、指标删除、替换、降权或 R2 参数冻结。输出不得包含 `best_indicator`、`best_layer`、`best_config`、`winner`、`optimized_q`、`freeze_candidate`、`review_candidate` 或 `do_not_freeze`。

## 指标 Registry 与 Grid

固定八项 active indicators：P1_NATR14、P2_LogRange20、C1_LogMASpread_5_60、C2_AdjVWAPSpread_5_60、T1_ER20、T2_AbsTrendT20、V1_TurnoverShrink20_60、V2_AmountLevel20Pct。V2 的 raw source 是 `V2_LogAmount20_base` / `LogAmount20`，active percentile 是 `V2_AmountLevel20Pct`，score 固定为 `1 - percentile`，不得对 V2 再计算一次 percentile。
