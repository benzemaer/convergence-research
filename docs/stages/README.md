# 研究阶段纲领稳定入口

`docs/stages/` 保存 R0–R6 每个研究阶段的阶段纲领稳定入口。阶段纲领用于说明阶段目标、边界、输入、输出、状态定义、阶段 task 路线图和下游交接；文件名使用稳定入口，不携带草案版本号，具体版本字段保留在文件正文中。

阶段纲领不替代 `docs/tasks/README.md`、具体 task 文件、run manifest、dataset manifest、artifact manifest、decision record 或 G0–G7 门禁证据。当前具体 task 只以 `docs/tasks/README.md` 为准；阶段纲领提供阶段级设计入口和下游交接上下文。

历史版本依赖 Git 历史追踪，不在本目录下创建 `archive/` 目录。实质性阶段纲领变更必须通过新的 task 或 PR 审核，并按项目变更控制规则记录影响范围。
